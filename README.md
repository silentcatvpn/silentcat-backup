# silentcat-backup
ابزار بک‌آپ خودکار Silent Cat Backup

این ابزار به شما کمک می‌کند تا به‌صورت خودکار از پنل مرزبان بکاپ گرفته و فایل را در تلگرام دریافت کنید
فرآیند نصب بسیار ساده است با فقط یک دستور:
bash <(curl -fsSL https://raw.githubusercontent.com/silentcatvpn/install-backup.sh/main/install-backup.sh) 

سپس فقط:

توکن ربات تلگرام خود را وارد کنید
شناسه چت (Chat ID) خود را وارد کنید
و ساعت بک‌آپ را مشخص کنید (عدد بین 1 تا 24)

✅ بعد از نصب، اسکریپت به‌صورت خودکار هر شب در ساعت انتخابی شما بک‌آپ می‌گیرد و برایتان ارسال می‌کند.

اگر در هنگام نصب یا استفاده از ابزار با مشکلی مواجه شدید با پشتیبانی در ارتباط باشید
